# Config

Configuracao da aplicacao (modelos de configuracao, leitura e validacao).

Arquivo de usuario bootstrapado automaticamente:

- `~/.asxrun/config`
- `~/.asxrun/history`

Chaves suportadas:

- `ollama_host` (URL absoluta, exemplo: `http://127.0.0.1:11434/`)
- `default_model` (modelo padrao para `ask/chat/skill`)
- `prompt_timeout_seconds`
- `healthcheck_timeout_seconds`
- `models_timeout_seconds`
